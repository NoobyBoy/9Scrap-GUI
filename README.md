# 9Scrap-GUI

A GUI application to display data scraped from 9gag. This project provides a user-friendly interface to visualize and analyze statistics from 9gag content that has been collected through web scraping.

## Overview

9Scrap-GUI is designed to help users explore and understand trends, patterns, and insights from 9gag's vast collection of memes and posts. By scraping data from 9gag and presenting it through an intuitive graphical interface, users can:

- View post statistics and engagement metrics
- Analyze trending topics and categories
- Track post performance over time
- Discover popular content patterns

## Features

- **Data Visualization**: Interactive charts and graphs displaying scraped 9gag data
- **Statistics Dashboard**: Comprehensive overview of post metrics, engagement rates, and trends
- **Category Analysis**: Breakdown of content by different categories and tags
- **Time-based Filtering**: View data from specific time periods
- **Search and Filter**: Find specific posts or content based on various criteria

## How It Works

1. Data is scraped from 9gag using web scraping techniques
2. The collected data is processed and stored
3. The GUI application loads and displays the data in various formats
4. Users can interact with the visualizations to explore the data

## Getting Started

### Prerequisites

- Python 3.7 or higher
- pip (Python package manager)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/NoobyBoy/9Scrap-GUI.git
   cd 9Scrap-GUI
   ```

2. Install the required dependencies:
   ```bash
   pip install -r requirement.txt
   ```


### Running the Application

Start the GUI application by running:
```bash
python main.py
```

The application will open in your default web browser at `http://localhost:xxxx` (the exact port will be displayed in the terminal).
